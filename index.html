🗄️ Step 1 — Choose a Table

Select which table you want to look at. Each table stores a different type of information.

📋 Step 2 — Choose Columns (leave all checked to get everything)

Tick only the columns you want to see in your results. Checking all = SELECT *

🔍 Step 3 — Filter Rows (optional)

Add conditions to show only specific rows. Example: show only students with grade = "A".

↕️ Step 4 — Sort Results (optional)

Choose a column to sort by and the direction — Ascending (A→Z / 0→9) or Descending (Z→A / 9→0).

🔢 Step 5 — Limit Rows

How many rows should be returned? Drag the slider to change the limit.
